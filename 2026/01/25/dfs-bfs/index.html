深度优先搜索（DFS）

在我系统学习树和图的存储方法之前，深度优先搜索对我来说，就是一种更科学的，可以自己控制条件的枚举方式。事实上它也部分符合这个定义。比起枚举，它能够模拟“先走到底，再换路线”的方式，枚举所有可能性。

以目前我的需求来讲，DFS分为三种：子集，排列，二维，接下来我会对每一个部分做简要介绍。
子集型
引入：洛谷P2036 PERKET

选了一个不是很若只的但是也不是很难的 ^^
可以得见，本题要求我们没有数量限制与顺序限制（不一定连续）地任意选配料，也就是要求我们选出它全部选项中最优的一个子集，这个就叫选子集的DFS形式。
首先，请我们用手自己比划一下，如果你想选完它所有的子集，你会怎么选？就我来讲，我会用左手大拇指按住第一个，右手食指开始点。一，一二，一三，一四，一五，一二三，一二四……一直到最后的五。除了顺序不同，其实这就和深度优先的步骤非常相似。

值得一提的是，在这里所选的子集实际上并不是只对他们执行“选”的操作，实际上在每一个子集中，每一个数字都会被执行一次操作，没有被选上的只是被执行了“不选”，比如选一三，实际操作它就是递归中一个“是否是否否”的分支。它和全排列的区别就是它可以有不选的权利。

P1219 八皇后问题

花了一天多从完全不懂到一遍过，嘻嘻
实际上蓝桥C/C++省A的第二题大多数都是由这个模板进化而来了。比如上一届考了一个不能连三个的，还有前面五子棋的，都是这个思路。相比单纯的枚举，dfs的优势在于它的限定条件是严格按照题目给定的，逻辑也比较清晰。其中if( r[i] || dg[x+i] || udg[(x-i+n)] ) continue; 这段根据题目条件设定限制条件的语句就是剪枝，可以优化速度。（没有听起来那么高级）

关于正对角线和反对角线，正对角线是右上对左下，反对角线是左上到右下，他们这个坐标表示是规律总结的，具体来说，也就是同一正对角线上的点横坐标和纵坐标和总是相等，同一反对角线就是差相等。但是差会是负数，所以需要加n，就必然是同一个正数了。
两者的尺寸本应均是棋盘n的最大值的两倍最稳妥，但是这题数据比较小，N开的大，就过了。标准上应该是2*N。

广度优先搜索（BFS）

如果深度优先搜索用的是栈（递归）来解决，那么广度优先用的就是队列。我们都知道BFS的搜索方式是一层一层扩散的，这其中实现“一层一层”就是用的把一层的每一个点推入队列，然后延伸计算。

多起点问题
P1332 血色先锋队

看题面真乐了
多起点其实和前一个题没有什么本质的区别，也就是起点需要先推到队列里罢了。注意这里要求碰到输出领主的最短时间，实际上第一个碰到的就是最短时间，所以剪枝的时候就是访问过的不能走就行。

#include<iostream>
#include<queue>
#include<cstring>

using namespace std;

const int N=510;
int n,m,a,b;
int d[N][N],dir[5]={-1,0,1,0,-1};
queue<pair<int,int> > q;
queue<pair<int,int> > l;


void bfs(){
    while(!q.empty()){
        auto t=q.front();
        int i=t.first,j=t.second;
        q.pop(); //要删点噢
        for(int k=0;k<4;k++){
            int dx=i+dir[k],dy=j+dir[k+1];
            if(dx>0 && dx<=n && dy>0 && dy<=m && d[dx][dy] == -1){ //是的，加上d[dx][dy] != -1是可以的
                d[dx][dy] = d[i][j]+1;
                q.push({dx,dy});
            }
        }
    }
}

void push(){
    while(!l.empty()){
        auto t=l.front();
        int i=t.first,j=t.second;  //这里就是领主的点要多次输出，就用一个push比较简洁。
        printf("%d\n",d[i][j]);
        l.pop();
    }
}

int main(){
    cin>>n>>m>>a>>b;
    memset(d,-1,sizeof d); //要初始化噢
    while(a --){
        int x,y;
        cin>>x>>y;
        d[x][y] = 0;
        q.push({x,y});
    }
    while(b --){
        int x,y;
        cin>>x>>y;
        l.push({x,y});
    }
    bfs();
    push();
    return 0;//看着自己以前望而却步的题目现在一遍过 真的很有成就感 
}
